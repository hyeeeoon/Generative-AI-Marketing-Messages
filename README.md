# 생성형 AI 기반 개인화 마케팅 메시지 자동 생성기

일경험사업(프로젝트형)

□ 프로젝트 과제 : 생성형 AI 기반 개인화 마케팅 메시지 자동 생성기
- 참여기간 : 2025년 10월 27일(월)~12월 19일(금)

## 📖 개요

고객의 이용 패턴·결제 이력·행동 로그 등 데이터를 바탕으로, 세그먼트별(또는 개인별)로 최적화된 마케팅 메시지를 생성형 AI로 자동 작성하고, 발송·효과 분석까지 연계하는 웹 서비스입니다.
목표는 운영자는 안정적·일관된 규칙(금지어/정책)을 유지하면서 고객사는 브랜드 톤·캠페인 목적에 맞춰 메시지를 세밀히 커스터마이징할 수 있도록 하는 것입니다.
